# Changelog

All notable changes to the Fimod VS Code extension are documented here.

The format is based on [Conventional Commits](https://www.conventionalcommits.org/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-beta.2] — 2026-04-25

### Features

- **playground:** add Playground panel
- **binary:** add auto-update check and bump esbuild to 0.25.0

### Documentation

- **release-workflow:** add notes/release-workflow.md reference doc
- **changelog:** convert CHANGELOG.md to Conventional Commits format

## [0.1.0-beta.1] — 2026-04-12

### Features

- **shape:** `Fimod: Shape` command with quick pick (molds + inline Python expressions)
- **shape:** diff preview with toggleable preview/format modes
- **shape:** recent history for molds and expressions
- **registry:** Registry Explorer sidebar view (sources, priorities, setup)
- **molds:** Local Molds sidebar view (open, run, test, details)
- **molds:** mold detail panel with README, source, mini playground
- **ui:** status bar indicator with fimod version
- **install:** auto-download of the fimod binary
