# Changelog

All notable changes to the Fimod VS Code extension are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-beta.1] - 2026-04-12

### Added

- `Fimod: Shape` command with quick pick (molds + inline Python expressions)
- Diff preview with toggleable preview/format modes
- Recent history for molds and expressions
- Registry Explorer sidebar view (sources, priorities, setup)
- Local Molds sidebar view (open, run, test, details)
- Mold detail panel with README, source, mini playground
- Status bar indicator with fimod version
- Auto-download of the fimod binary
